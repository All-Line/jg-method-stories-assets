# JG Method — Stories Assets

Public asset host for JG Method daily editorial Stories content.

**Purpose:** Each editorial folder contains the 7 PNG slides + captions for daily JG Method Stories. URLs in the format `https://raw.githubusercontent.com/All-Line/jg-method-stories-assets/main/<folder>/<file>` are consumed by social scheduling tools (Metricool API) to schedule TikTok Photo Mode and other formats.

**Owned by:** Mino Mazzamati
**System:** JG OS 2.0 — automated daily SOP `jg-stories-editorial-diario`

## Folder convention

`jg-stories-<theme>-YYYY-MM-DD/`
- `jg-stories-<theme>-01.png` to `-07.png` — 7 Story slides (1080×1920)
- `captions.md` — Instagram + TikTok captions
